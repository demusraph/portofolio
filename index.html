04
Body Fat Prediction — KNN Regression
A machine learning model using K-Nearest Neighbors Regression to accurately predict body fat percentage from simple anthropometric measurements, achieving R² = 0.9277 with RMSE of 1.83.
PythonKNNscikit-learnData Analytics
VIEW DECK
+
OVERVIEW
Final project for Dasar Analitik Data course (Dr. Arian Dhini). Used the Body Fat Prediction Dataset (252 adult men, 15 variables) from Kaggle. Applied full ML pipeline: EDA with correlation heatmaps, StandardScaler preprocessing, 80/20 train-test split, and K-value optimisation.
KEY OUTCOMES
  • Best K = 4 · R² = 0.9277 (highest among K=1–19)
  • RMSE = 1.8334 · MAPE = 11.3% — reliable prediction accuracy
  • Abdomen (0.81) & Density (−0.99) identified as top predictors
  • Bonus: 5-tier body fat classification (Athletes → Obesity)
